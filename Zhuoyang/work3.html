Tools and Technologies

HardWare Part

Since our project is mainly for students from the sixth grade to the 12th grade, we must reduce the hardware requirements as much as possible. All we hope that our project can be easily completed by a notebook or mobile terminal such as a mobile phone.

SoftWare Part

Since our project is aimed at this large group of students, we also want to talk about the minimum requirements in terms of software

  • First of all, the current mainstream Windows system should be able to easily run this project. For example, currently has a Windows7 system with a 47% utilization rate, and an increasingly popular WINDOWS10 system

    • Since our project is a program to help students answer questions, we need to obtain students' questions through different methods. For example, students can enter the questions they want to ask in the question box by typing, and the system will automatically analyze these questions and find the corresponding answers. And according to the age of the students who ask the question, choose the corresponding answer method to ensure that all students can get the answers they need
